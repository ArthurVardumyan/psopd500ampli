Чтобы изменить частоту данных (samples rate) необходимо в файле \psopd2v\Psopd\Inc\defines.h изменить значение макроса #define ODR на одно из значений: 1000 или 500. Откомпилировать проект в среде IAR EWARM. Далее, используя DfuSe получить файл образа .dfu и загрузить его в контроллер средствами DfuSe и встроенного загрузчика.
